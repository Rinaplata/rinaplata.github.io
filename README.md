# Rina Plata | Portafolio de ingeniería de software, frontend y comunidad

Portafolio personal de **Rina Plata**, ingeniera de software, desarrolladora frontend, speaker y líder comunitaria indígena Wayuu de Colombia. Este sitio presenta experiencia profesional, proyectos, charlas, comunidades, habilidades técnicas y trabajo de impacto social desde la tecnología.

## Sobre Rina Plata

Soy ingeniera de software y desarrolladora frontend enfocada en crear experiencias web modernas, accesibles y humanas. Mi trabajo conecta **Angular**, **React**, **TypeScript**, **accesibilidad web**, **comunidades tecnológicas**, **liderazgo femenino** e **identidad indígena Wayuu**.

Desde Colombia participo en espacios de tecnología, educación e inclusión, con interés especial en productos digitales accesibles, comunidades indígenas, mentoría y construcción de software con impacto social.

## Ejecutar localmente

Instalar dependencias:

```bash
npm install
```

Levantar servidor de desarrollo:

```bash
npm start
```

Abrir:

```txt
http://127.0.0.1:4200/
```

## Build de producción

```bash
npm run build
```

El resultado se genera en:

```txt
dist/rinaplata-portfolio/browser
```
